## pokemon

## Features
### List all Pokémon
### Show paginated results
### Detail view of Pokémon
### Page sizer with options

## Usage
1. Run the application
`npm install`

`npm start`
Open http://localhost:3000 to view in the browser

2. Run the tests 
`npm run test`


3. Build application
`npm run build`
