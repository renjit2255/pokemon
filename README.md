## pokemon

### Features
1. List all Pokémon
2. Show paginated results
3. Detail view of Pokémon
4. Page sizer with options

### Todo Features
1. Search
2. Sorting

### Usage
1. Run the application
`npm install`

`npm start`

Open http://localhost:3000 to view in the browser

2. Run the tests 
`npm run test`


3. Build application
`npm run build`
